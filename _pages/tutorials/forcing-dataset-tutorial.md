### Overview

We have prepared a series of Python scripts to help the user to preprocess forcing dataset required to run the model. These scripts can be used to download and preprocess precipitation and potential evapotranspiration. The datasets used in this example are only IMERG, ERA5, and hPET, but the user can use any other dataset to run the model.

The user are welcome to download and edit any of the materials at their own responsability. The CUWALID team is not responsible for any of the outputs generated from any of the CUWALID models or related scripts.
