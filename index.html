CUWALID Input Helper

Welcome to the CUWALID Input Helper! This tool allows you to easily generate JSON configurations for running the different aspects of CUWALID.

For running the main CUWALID system, you will need all of the json files. It is important to note that when running the system, the CUWALID input json file will override the necessary settings within each model. For example, if 'year' is set to '2025' in the CUWALID json, but '2017' in the storm input, the storm input will be overrided.

How it works:

  • Select the component you would like the input for at the top.
  • Fill out the form fields as required.
  • Required fields are marked in red, while optional fields are in black.
  • Once completed, you can view the preview of your json and export it.
